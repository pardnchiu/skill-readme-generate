## Features

- **Bilingual Output**: Generates both `README.md` (English) and `README.zh.md` (Traditional Chinese)
- **Source Code Analysis**: Automatically parses project structure to extract type definitions, function signatures, and dependencies
- **Multi-Language Support**: Supports Go, Python, JavaScript, TypeScript, PHP, and Swift projects
- **LICENSE Generation**: Supports MIT, Apache-2.0, GPL-3.0, BSD-3-Clause, ISC, and Unlicense license types
- **Flexible Parameters**: Specify public/private mode, custom repo path, and license type
- **Automatic Badge Generation**: Generates appropriate badges based on project language

## Usage

### Basic Syntax

```bash
/readme-generate [private] [LICENSE_TYPE] [REPO_PATH]
```

### Parameters

| Parameter | Format | Example | Description |
|-----------|--------|---------|-------------|
| `private` | Keyword | `private` | Generate without badges and star history |
| `LICENSE_TYPE` | License identifier | `MIT`, `Apache-2.0` | Generate LICENSE file |
| `REPO_PATH` | `github.com/{owner}/{repo}` | `github.com/foo/bar` | Override default owner/repo |

### Examples

```bash
# Generate README only (public mode)
/readme-generate

# Generate README + MIT LICENSE
/readme-generate MIT

# Private mode (no badges or star history)
/readme-generate private

# Private mode + MIT LICENSE
/readme-generate private MIT

# Specify custom repo path
/readme-generate github.com/foo/bar

# Full parameters
/readme-generate private MIT github.com/foo/bar
```

## CLI Reference

### Supported License Types

| Type | Aliases (case-insensitive) |
|------|----------------------------|
| MIT | `mit` |
| Apache-2.0 | `apache`, `apache2`, `apache-2.0` |
| GPL-3.0 | `gpl`, `gpl3`, `gpl-3.0` |
| BSD-3-Clause | `bsd`, `bsd3`, `bsd-3-clause` |
| ISC | `isc` |
| Unlicense | `unlicense`, `public-domain` |
| Proprietary | `proprietary` (auto-enables private mode) |

### Analysis Script

```bash
python3 scripts/analyze_project.py /path/to/project
```

Outputs JSON containing:
- `language`: Project language
- `name`: Project name
- `version`: Version number
- `types`: Type definitions
- `functions`: Function signatures
- `dependencies`: Project dependencies
