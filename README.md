# readme-generate

[![license](https://img.shields.io/github/license/pardnchiu/readme-generate)](LICENSE)
[![version](https://img.shields.io/github/v/tag/pardnchiu/readme-generate?label=release)](https://github.com/pardnchiu/readme-generate/releases)

> Automated bilingual README generator that analyzes source code to produce professional README.md (English) and README.zh.md (Traditional Chinese) documentation.

## Features

- **Multi-language project analysis**: Supports Go, Python, JavaScript/TypeScript, PHP, and Swift projects
- **Automatic type extraction**: Extracts struct, class, and interface definitions from source code
- **Function signature capture**: Identifies exported functions and extracts complete signatures
- **Bilingual output**: Generates both English and Traditional Chinese README versions
- **Smart ignore mechanism**: Automatically skips `node_modules`, `vendor`, `__pycache__`, and similar directories
- **Dependency analysis**: Extracts dependencies from `go.mod`, `package.json`, and `pyproject.toml`

## Installation

```bash
# Copy this skill to the Claude Code skills directory
cp -r readme-generate ~/.claude/skills/
```

## Usage

Run in Claude Code:

```bash
# Basic usage - generate README only
/readme-generate

# Generate README + MIT LICENSE
/readme-generate MIT

# Private mode (no badges and star history)
/readme-generate private

# Specify custom GitHub path
/readme-generate github.com/your-org/your-repo

# Combined usage
/readme-generate private MIT github.com/your-org/your-repo
```

### Supported LICENSE Types

| Type | Aliases |
|------|---------|
| MIT | `mit` |
| Apache-2.0 | `apache`, `apache2`, `apache-2.0` |
| GPL-3.0 | `gpl`, `gpl3`, `gpl-3.0` |
| BSD-3-Clause | `bsd`, `bsd3`, `bsd-3-clause` |
| ISC | `isc` |

| Proprietary | `proprietary` (auto-enables private mode) |

Standalone script for analyzing project structure.

  "version": "1.0.0",
